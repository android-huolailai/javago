
## 1. 集合的处理
```java
    /**
     *  向某个list添加数据
     * @param list 目标list
     * @param data 待添加的data
     * @param isSame 是否相同
     * @param <T> 数据类型
     * @return
     */
    public static <T>List<T> addTToList(List<T> list, T data, Predicate<T> isSame) {
        list.removeIf(isSame);
        list.add(data);
        return list;
    }
```
